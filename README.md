# VaultIX

**Store smarter. Find faster.**

VaultIX is a premium, cinematic-first vaulting ecosystem designed for high-performance data management and visualization. Built with a focus on immersive user experience, VaultIX combines 4K visual storytelling with robust cloud-native architecture.

---

## 🚀 Vision: The Cinematic Vault

VaultIX transforms the traditional "dashboard" into a cinematic experience. The platform features:
- **4K High-Resolution Identity**: An immersive 4K video gateway (`ex2.mp4`) that sets a premium tone from the first interaction.
- **Adaptive Responsive Strategy**:
    - **Case 1 (Big Screens)**: Clean, cinematic purity focused on the high-fidelity video with minimalist overlays.
    - **Case 2 (Mobile)**: Intelligent cropping focused on the 3D vault core with high-definition manual branding.

## ✨ Key Features (95% Complete)

### 🔐 Secure Entry & Identity
- **Supabase-Powered Auth**: Robust, secure authentication and session management.
- **Persistent Preferences**: User-specific settings (Theme, View Mode, Sort Order) are saved to cloud metadata for a consistent experience across all devices.

### 📊 Intelligent Analytics Dashboard
- **Data-Driven Insights**: Interactive charts and automated data visualizations for vault health and activity.
- **Professional SaaS Aesthetic**: A refined, high-contrast dark mode design system built for professional use.

### 📁 Advanced Vault Management
- **3D Vault Interaction**: High-fidelity 3D vault core animations.
- **Sophisticated Organization**: Persistent view toggles and multi-level sorting logic.

### ⚙️ Account & Privacy
- **Granular Controls**: Display Name and Email management.
- **Privacy First**: Integrated bug reporting and complete account data deletion functionality.

## 🏁 Getting Started

### Prerequisites
- Node.js 18+
- Supabase Account (for Auth & Database)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/vaultix.git
   cd vaultix
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env.local` file with your Supabase credentials:
   ```env
   NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
   NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) to experience VaultIX.
